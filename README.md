📊 Sales Performance Dashboard | Power BI
Project Overview

This project presents an interactive Sales Performance Dashboard built in Power BI to analyze business performance across sales, customers, products, and regions. The dashboard transforms raw sales data into meaningful insights through interactive visualizations, KPIs, and dynamic reports, enabling data-driven decision-making.

Key Features
Cleaned and transformed data using Power Query.
Built an optimized data model by creating relationships between multiple tables.
Developed DAX measures and calculated columns for KPIs and dynamic calculations.
Designed interactive dashboards with cards, charts, slicers, filters, and drill-through functionality.
Tracked key business metrics including Total Sales, Profit, Orders, Quantity Sold, and Average Order Value.
Analyzed customer behavior, regional performance, product categories, and sales trends.
Created an intuitive and user-friendly dashboard to support business intelligence and decision-making.
